Este programa sirve para guardar y organizar varias películas.
Cada vez que el usuario agrega una película, el programa la escribe en un archivo de texto con su nombre, genero y año.
Ademas, tambien guarda algunos datos numéricos (como popularidad y rating) en un archivo binario.
El menú permite noa agregar nuevas películas, ver toda la lista, buscar una por su nombre y ver los datos binarios.
También usa excepciones para evitar errores, 
por ejemplo cuando el archivo no existe o cuando el usuario mete un dato incorrecto.
